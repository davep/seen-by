About this site

This is a "rescued" version of the photoblog that I started on Blogger.com back in 2016. This in turn was a reboot of an earlier photoblog that I used to maintain, which was long ago lost to the whims of tech buyouts (which is backed up on WordPress and which I might try and export and pull into here at some point).

The rescue came about due to the fact that I nuked my Twitter account quite some time ago which broke the whole posting/hosting process. While I did keep backup copies of all the images, there was no simple way to revive the blog.

So this is an archive that I fully control.
